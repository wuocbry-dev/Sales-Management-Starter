# VN Premium SaaS Light (2025)

## Brand
- Trustworthy, efficient, modern, friendly for non-technical Vietnamese SME users.
- Premium but approachable; prioritize readability over flashy effects.

## Visual language
- Light dashboard with spacious layout, clear hierarchy.
- Soft blue primary (#2F6BFF), subtle teal accents (#14B8A6) for highlights/status.
- Large rounded cards (12px), thin borders (1px), soft shadows.
- Very light glassmorphism only for header/search surfaces if useful.

## Components
- Polished metric cards with icon + trend pill.
- SaaS tables: sticky header feel, hover rows, status pills.
- Charts: area + bar + donut; muted grids; strong labels.

## Data formatting
- Currency: prefix ₫ and dot separators (e.g., ₫128.450.000).
- Vietnamese labels; concise statuses.
